# Projeto Integrador - Análise de Vagas em Creches de Marília/SP

## Descrição

Este projeto tem como objetivo analisar a demanda por vagas em creches municipais para crianças de 0 a 1 ano na cidade de Marília/SP.

## Problema

Muitas famílias enfrentam dificuldades para conseguir vagas em creches municipais, gerando impactos sociais e econômicos.

## Objetivos

- Identificar bairros com maior déficit de vagas.
- Calcular indicadores de atendimento.
- Produzir análises para apoio à tomada de decisão.

## Tecnologias previstas

- Python
- Pandas
- SQL
- Power BI
